مشروع: نظام إدارة العملاء والطلبات والمخازن المتكامل مع Google Sheets (صفحتان فقط)

- واجهة أمامية (React.js + Tailwind CSS/Material UI)
- سيرفر خلفي (Node.js + Express.js)
- مصادقة (Google OAuth/Firebase Auth)
- إدارة جلسات (Redis)
- تكامل مباشر مع Google Sheets API
- Google Sheet واحد فقط، يحتوي على:
    - Sheet 1: جداول (العملاء، الطلبات، المقاسات، التركيبات)
    - Sheet 2: جدول المخزون
    - مع إمكانية إضافة جداول جديدة داخل كل Sheet عند التوسع
- نظام إشعارات داخلي يربط بين الأقسام، مع تسجيل حالة الإشعار في كل جدول (NotificationSent)
- دعم رفع ملفات PDF، تخصيص النماذج، رسائل منبثقة
- قابلية التوسع وإضافة أقسام مستقبلية
- ألوان راقية (بني، بيج، أبيض)
- دعم تعدد المستخدمين والصلاحيات
- واجهة تحكم للمدير لتخصيص كل شيء
- إمكانية النشر السريع والتوسع عبر Docker

ملاحظة: جميع البيانات مخزنة في Google Sheet واحد فقط، وكل قسم له جدول منفصل داخل إحدى الصفحتين.
